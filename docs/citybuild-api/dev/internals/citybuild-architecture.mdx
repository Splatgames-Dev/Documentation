# CityBuild Architektur

Die CityBuild Architektur ist sehr komplex und besteht aus mehreren Komponenten, die miteinander interagieren.
Die wichtigsten Komponenten auf die der Kernel aufbaut sind:

- ProtocolLib
- PlaceholderAPI

Diese Komponenten sind für die Kommunikation zwischen dem Server und dem Client verantwortlich.
Sie ermöglichen es, dass der Server mit dem Client kommunizieren kann und umgekehrt.

## ProtocolLib

ProtocolLib ist ein Plugin, das es ermöglicht, mit dem Client zu kommunizieren.
Es ermöglicht es, Daten zwischen dem Server und dem Client zu übertragen und zu empfangen.
Dies ist notwendig, um die Funktionen des CityBuild Servers zu implementieren.

## PlaceholderAPI

PlaceholderAPI ist ein Plugin, das es ermöglicht, Platzhalter in Nachrichten zu ersetzen.
Dies ist notwendig, um die Funktionen des CityBuild Servers zu implementieren.
