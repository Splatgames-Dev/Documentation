# Willkommen bei der CityBuild API

Die CityBuild API ist die API, mit der Entwickler auf interne Funktionen des CityBuild Plugins zugreifen können.
Diese API ermöglicht es, eigene Funktionen zu erstellen, die auf die Datenbank zugreifen und mit dem CityBuild Plugin interagieren.

Die CityBuild-API ist in verschiedene Module unterteilt, die jeweils unterschiedliche Funktionen bereitstellen.
Diese API macht ausschließlich Zugriffe auf die internen Daten und ermöglicht es somit Entwicklern mit dem CityBuild Plugin zu interagieren, unabhängig von internen veränderungen.
